# Spring Boot Demo

Este proyecto es un ejemplo simple que demuestra cómo configurar y utilizar Spring Boot. Está diseñado para ayudarte a comprender los conceptos básicos y las funcionalidades principales del framework.

## Características
- Configuración inicial de una aplicación Spring Boot.
- Ejemplo básico de controladores.
- Introducción a los servicios y repositorios.

Este proyecto forma parte de tu ruta de aprendizaje de Spring Boot. ¡Explora, experimenta y disfruta aprendiendo!
